# eating-game
A simple browser game where the player (plate) moves left/right to catch falling food. Built with plain HTML, CSS and JavaScript. Folders are named `html`, `css`, and `javascript` as requested.

### How to run
1. Clone this repository or download the files.
2. Open `html/index.html` in a browser.

### Controls
- Arrow Left / Arrow Right — move
- Space / Up Arrow — pause / resume

### Notes
- Uses emoji for food and player so there are no external assets.
- Easy to extend with images or levels.
